What's new in Nexus

A chronologically ordered log of new features, core gateway improvements, and critical security patches shipped into production.

Version 1.4.2

July 15, 2026
Security

Enhanced Asymmetric Cipher

Upgraded the vault encryption mechanism for upstream Groq keys. Credentials are now masked using a deeper block cipher standard before ever reaching the edge routing layer.

Fix

Dashboard Skeleton Jitter

Resolved an issue where DOM rendering caused abrupt flashes during data fetching. Integrated a 0ms-to-250ms opacity transition matrix across all API key and usage metric components for smooth skeleton replacements.

Improvement

Global Modal Override

Replaced standard synchronous browser popup alerts with a globally injected, non-blocking custom promise-based modal ecosystem for continuous native UI harmony.

Version 1.4.0

July 02, 2026
Feature

Sister Lab Project Spotlight

Introduced the Builder page showcasing "Jarvis AI Assistant", integrating visibility into our standalone hybrid Python entity operating on Picovoice biometrics and Gemini logic cores.

Improvement

Silicon Valley Layout Paradigm

Overhauled the entire presentation layer from 1200+ lines of custom CSS to an optimized Tailwind-driven framework. Introduced deeper blacks, sharper glassmorphism, and radial glows.

Version 1.2.0

May 14, 2026
Feature

Dynamic CORS Whitelisting

Deployed origin host verification. Requests originating from unauthorized URLs without a matched domain in the user vault are now actively dropped at the edge to prevent token manipulation.

Fix

Firebase Auth Sync Issue

Patched a race condition where the dashboard would redirect authenticated users to login prior to the JWT session state resolving across rapid page reloads.

Version 1.0.0

January 10, 2026
Launch

Nexus Architecture Genesis

Nexus officially launched its core beta. Initializing asynchronous Groq client generation, client-side script distribution capabilities, and zero-friction UI mapping. Welcome to the future of frontend AI security.
