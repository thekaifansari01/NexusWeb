Nexus Web Assistant v3.0.0

Turn any webpage into a conversational knowledge base. Embed in seconds, secure by design.

Nexus Web Assistant is a lightweight, embeddable AI chat widget that automatically scrapes your page content, understands the context, and answers user questions using Groq's ultra-fast LLMs — all while keeping your API keys secure through encryption and domain whitelisting.

Security First — Your Groq key is never exposed.

You store your Groq API key in the encrypted dashboard vault. The widget uses a separate, domain-restricted Nexus key for communication. Our proxy validates the domain, decrypts your Groq key, and forwards requests — your key never touches the browser.

🚀 Quick Start

Embed the Nexus widget on your website in two simple steps.

1

Get your Nexus API Key

Sign in to the Nexus Dashboard with Google. In the Nexus Keys tab, create a new key and copy it. Then, in the Settings tab, store your Groq API key — it will be encrypted and never exposed.

2

Add the script tag

Paste the following code into your HTML, preferably just before the closing </body> tag.

<script>
window.NexusConfig = {
  apiKey: 'YOUR_NEXUS_API_KEY',
  model: 'meta-llama/llama-4-scout-17b-16e-instruct',
  botName: 'Nexus AI',
  greeting: '👋 Hello! How can I assist you today?'
};
</script>

<script defer src="https://cdn.jsdelivr.net/npm/nexus-web-assistant@3.0.0/dist/nexus-assistant.min.js"></script>

That's it! Your website now has an AI assistant that understands its content. Users can ask questions and get instant, context-aware answers.

⚙️ Configuration Options

Customize the widget's behaviour by setting the window.NexusConfig object before the script loads. All properties are optional except apiKey.

Property Type Default Description
apiKey string '' Required. Your Nexus API key from the dashboard.
model string 'llama-4-scout-...' Groq model to use. Supported: llama-4-scout-17b-16e-instruct, llama3-8b-8192, mixtral-8x7b-32768.
botName string 'Nexus AI' Display name of the assistant shown in the chat header.
greeting string Default Initial bot message shown when the panel opens. Supports Markdown.
systemPrompt string Default Override the system instructions. Page context is automatically injected.
theme string 'auto' 'dark', 'light', or 'auto'.

Example Configuration: 

window.NexusConfig = {
  apiKey: 'nxs_abc123def456',
  model: 'llama3-70b-8192',
  botName: 'DocBot',
  greeting: '📚 Welcome to our documentation! Ask me anything.',
  systemPrompt: 'You are a helpful documentation assistant. Always cite the source.',
  theme: 'dark'
};

System Prompt Tip: Your custom prompt is appended to the page context. The AI will always prioritize the page content over general knowledge.

🎨 Theming

Nexus automatically adapts to your website's theme, but you can also force a specific mode or let users toggle it.

Auto-detection

By default (theme: 'auto'), the widget checks in this order:

  1. Your window.NexusConfig.theme override
  2. Saved user preference in localStorage
  3. dark/light class on <body> or <html>
  4. data-theme attribute on <html>
  5. System preference via prefers-color-scheme
  6. Defaults to dark

Pro tip: The widget also respects <meta name="theme-color"> brightness to infer your site's preferred theme dynamically.

✨ Key Features

Context Scraping

Extracts clean Markdown from your page content — headings, paragraphs, lists, and code snippets. The AI answers with full context.

File Attachments 5MB max

Users can attach images, PDFs, text files, and Word documents securely via the widget interface.

SPA Support

Automatically detects navigation changes in single-page apps (pushState / popState) and refreshes the page context dynamically.

Smart Caching 24h TTL

Scraped content is cached in IndexedDB. Repeat visitors get instant responses without re-scraping.

🔐 Security Model

Nexus is built with a zero-trust approach to API key management. Here's how your keys are protected:

1

You store your Groq API key in the Dashboard

Your key is encrypted with AES-256 before being saved to Firestore. Only you can view or delete it.

2

You generate a Nexus API key for your frontend

This key is safe to embed in your HTML. It is domain-restricted — it only works for origins you authorize.

3

The widget sends your Nexus key to our proxy

The proxy validates the domain, decrypts your Groq key, and securely forwards the request to Groq's servers.

Programmatic Control

The widget exposes a global API for advanced use cases. Call these functions from your own JavaScript:

Method Description
window.openNexus() Opens the chat panel programmatically.
window.closeNexus() Closes the chat panel.
window.toggleNexus() Toggles the chat panel open/closed.
window.refreshNexusContext() Forces a re-scrape of the page content. Useful after dynamic DOM updates.

🔌 API Reference

The widget communicates with your backend proxy at POST /api/chat. The expected payload structure is:

{
  "nexusKey": "nxs_abc123def456",
  "messages": [
    { "role": "system", "content": "You are a helpful assistant..." },
    { "role": "user", "content": "What is this page about?" }
  ],
  "model": "meta-llama/llama-4-scout-17b-16e-instruct"
}

You can find the serverless function implementation in api/chat.py in the open-source repository.
