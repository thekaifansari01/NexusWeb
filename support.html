Frequently Asked Questions

Quick answers to common integration questions.

What is Nexus?

Nexus is a secure, drop-in AI assistant that turns any static website into a conversational knowledge base using Groq's ultra-fast Inference Engine. It's built for scale and takes minutes to deploy.

How does domain whitelisting work?

Domain whitelisting ensures only authorized domains can utilize your generated Nexus API keys. Our edge gateway enforces strict origin verification on every incoming request, instantly blocking unauthorized cross-origin requests.

Is my core API key secure?

Absolutely. Your master Groq key is heavily encrypted (AES-256) and stored securely in our backend vault. It never reaches the client-side. The frontend widget operates purely on a separate, domain-restricted proxy key.

What models does Nexus support natively?

Nexus natively supports llama3-8b-8192 for extreme speed, llama3-70b-8192 for complex reasoning, and mixtral-8x7b-32768 for large context windows, all powered seamlessly by Groq.

Is Nexus actually free?

Yes. The Hobby tier provides 1,000 requests per month completely free, with no credit card required. As your traffic scales, you can easily upgrade to our Pro tier for enhanced limits.
